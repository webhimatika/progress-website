On Saturday (16/3) HIMATIKA has held Quality Time of HIMATIKA in Building A FMIPA UNS room A316 & A317. This activity was organized by the Institutional Consolidation division and was attended by all HIMATIKA FMIPA UNS 2024 management.

Because it coincides with the month of Ramadan, the first Quality Time event this time is by breaking the fast together and playing the game “Crossword” together. So each division works together to solve several series of questions that have been given.

The event started with the event begins with the MC giving the command to start from that number, then each division works together to find the right answer, when time is out, each division must submit the answer. After that the event continued with breaking the fast together. Then the event was closed with documentation.
